#### OriginsOnMars - это модпак на Forge 1.18.2 с Origins с технико-магической тематикой.
Развивайте техническую ветвь с *ComputerCraft*, *Create*, *Mekanism*, *Industrial_* или магическую ветвь с *Ars Nouveau*, *Enchanted: Witchcraft* и *Malum*.\
Модпак создавался только для сервера OriginsOnMars, но распространяется по лицензии Creative Commons Zero.

# Установка!!!
1. Скачать ZIP-архив последнего релиза модпака (справа, с зелёной пометкой "Latest").
2. Извлечь файлы из архива в папку вашей игры: `.minecraft` либо `.tlauncher/legacy/Minecraft/game`.

При повторной установке удалить файлы с прошлой установки!
